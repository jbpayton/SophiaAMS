# Memory Store

Save new information to your long-term semantic memory.

## Usage

```run
from sophia_memory import memory

result = memory.store("Joey loves Python programming")
print(result)
```

## When to Use
- When you learn something important that should be remembered
- When the user explicitly asks you to remember something
- When extracting key facts from conversations or web pages
